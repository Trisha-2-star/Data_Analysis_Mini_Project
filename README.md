## 📌 Overview
This project demonstrates core data analysis skills using SQL, Excel, and Python.
## 🗂 Project Structure
- **SQL/**
  - Hotel and Clinic schema queries
- **Spreadsheets/**
  - Ticket analysis using Excel
- **Python/**
  - Time conversion and duplicate removal scripts
## 🛠 Tools Used

- SQL (MySQL / PostgreSQL)
- Microsoft Excel
- Python
## 📊 SQL Tasks

- Retrieved last booked room per user
- Calculated billing amounts
- Filtered bills greater than 1000
- Identified most and least ordered items
- Found second highest bill per month
- Analyzed clinic revenue, profit, and top customers
## 📈 Excel Tasks

- Used **VLOOKUP** to fetch ticket created date
- Checked tickets closed on:
  - Same day
  - Same hour
- Used **IF, HOUR, COUNTIF** functions

## 🐍 Python Tasks

1. Converted minutes into hours and minutes  
2. Removed duplicate characters from a string using loops  

## ✅ Conclusion

This assignment showcases practical data analysis skills including data querying, data manipulation, and basic programming logic.
